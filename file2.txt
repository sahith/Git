Git commands
In order add the untracked files to the staging area the command used is
git add -A(This adds all the untracked files)

If you want a specific file to go inot the staging area then
git add <file-name>

After Add command your untracked files now will be in staging area

AFter add the command follows is commit
git commit -m "<your message appears here>"

Git log is used to view the commits

git log --oneline is used to list all the commits in one line

git log --oneline | wc -l is used to list the total number of commits
